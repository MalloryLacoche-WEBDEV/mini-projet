# Mini projet permettant de calculer les rapport de protéine journaliers pour différents poids.

>Ce projet permet de sélectionner un poids minimum et un poids maximum et d'afficher le nombre de gramme de protéine par jour.
---

## Fonctionnalités

Voici ce que permet de faire cette application :

* Choix du poids minimum et maximum : Permet de choisir un poids minimum et maximum, avec certaines conditions (min < max, uniquement des chiffres).
* Choix du nombre de lignes : Permet à l'utilisateur de choisir le nombre de lignes renvoyées dans le tableau des résultats.
* Export du résultat au format CSV ou PDF : Permet d'exporter le tableau de résultats au format PDF ou CSV.
